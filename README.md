# Ledis-cli

Simple web-based CLI

## Folder structure

```
    |-- public/
        |--css/
        |--js/
        |--templates/  // html file
    |--src/
        |--structures/  // data structure
        |--server.js // main js server
    ...
```

## How to run:

```

git clone https://github.com/burdy035/ledis-cli.git

cd ledis-cli

npm install

npm run serve

```
